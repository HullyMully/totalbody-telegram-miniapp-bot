# Screenshots

Approved portfolio screenshots are stored in the root-level `screens/` directory.

Before publishing or replacing screenshots, check that each image:

- is approved for portfolio use;
- does not show client names, phone numbers, Telegram IDs or chats;
- does not show private booking records;
- does not show admin-only links or panels;
- represents the TotalBody interface accurately.

The screenshots are included as portfolio evidence, not as a source of operational data.
