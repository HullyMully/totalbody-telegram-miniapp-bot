# Privacy

This repository is prepared as a public portfolio version of a freelance project for TotalBody. It keeps the brand context and approved portfolio screenshots, but excludes private operational data.

## Not Published

- Telegram bot tokens.
- API keys and private keys.
- `.env` files.
- Real admin Telegram IDs.
- Client Telegram IDs.
- Client phone numbers.
- Client names and booking records.
- Database files and dumps.
- Logs and exports.
- Private uploads or private photos.
- Internal admin links and live operational links.

## Environment Files

`.env` is not included because it can contain secrets and deployment-specific settings. The repository includes `.env.example` with safe placeholders only.

## Demo and Mock Data

The public version uses placeholders for secrets, private contact channels and live booking links. Real client records are excluded. Local demo storage can create `demo.db`, and database files are ignored by Git.

## Screenshots

Screenshots are used only as portfolio screenshots. Before publishing, each screenshot should be checked to confirm that it is approved and does not reveal client records, private chats, phone numbers, admin panels or unpublished media.

## Git History Note

If an existing local Git history contains private data, do not push it to a public repository. Publish from a cleaned public branch or a fresh public repository that contains only approved files.
